# 💰 Finance Dashboard UI

## 📌 Project Overview

This is a **Finance Dashboard UI** built using React.  
It helps users track financial activity, view transactions, and understand spending patterns.

---

## 🚀 Features

### 📊 Dashboard Overview

- Total Balance, Income, Expenses
- Interactive buttons to switch views

### 📈 Charts

- Line chart (Balance trend)
- Pie chart (Spending breakdown)

### 📋 Transactions

- View transaction list
- Search/filter by category

### 👤 Role-Based UI

- Viewer → can only view data
- Admin → can add transactions

### 💡 Insights

- Highest spending category
- Total income & expenses
- Financial observation

### 🌙 Dark Mode

- Toggle between light and dark themes

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript (ES6)
- CSS (Custom styling)
- Recharts (for charts)
